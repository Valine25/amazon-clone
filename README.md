# Amazon Clone (HTML + CSS)

This is a basic clone of the Amazon website built using only **HTML** and **CSS**.
It's a front-end only project, made for practice and learning layout design, responsive UI, and styling techniques.

##  Technologies Used

- HTML5
- CSS3

##  Features

- Amazon-style navigation bar
- Product cards
- Responsive layout (if media queries used)
- Footer section

##  How to Run

1. Clone the repository:git clone https://github.com/your-username/amazon-clone.git
2. Open `index.html` in your browser.
